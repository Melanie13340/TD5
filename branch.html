Commandes branch --set-upstream

Si la branche indiquée n'existe pas encore ou si "--force" a été utilisé, alors la commande "git branch --set-upstream" agit exactement comme "--track". Sinon elle reste fixe jusqu'à la configuration comme "--track" resterait lors de la création de la branche, sauf si les points de la branche n'ont pas été modifié .

Syntaxe

git branch --set-upstream

Affichage

* master

Commande branch --set-uptream-to

On utilise la commande "branch --set-upstream-to=" pour creer une nouvelle branche master d'un fork creer à partir d'une branche master locale.

Syntaxe

git branch --set-uptream-to=my_fork/master

Affichage

Branch gh-pages set up to track remote branch master from my_fork.
